USER GUIDE:

1. Upload a CSV dataset using the 'Browse Files' button.
2. Click the 'Analyse File' button underneath the raw dataset preview (if enabled).
3. Select a commodity from the dropdown menu (Total expenditure is the default commodity used for this project).
4. Ensure all filters you require are checked, and you have selected the years you want to predict (if applicable).
5. Click the 'Display Visualisation' button to see the visualisation of the data.
6. The visualisation will load and show the trend of the data you have selected as well as the predictions (if applicable).
